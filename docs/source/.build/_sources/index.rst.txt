.. note::
   Working document for srsRAN Project. This is still a work in progress 

To get started with srsRAN Project, take a look at the following guides: 

   - :ref:`Install<installation>` srsRAN Project on your computer.
   - Read the gNB :ref:`user manual<user_manual>`. 
   - Get up and running with srsRAN Project by checking out our :ref:`tutorials<tutorials>`. 
   - More advanced users should read the :ref:`Developers Guide <dev_guide>`.
   - Get up to speed on 5G and O-RAN with our :ref:`Knowledge Base <knowledge_base>`. 

Other useful resources include: 

    * The srsRAN Project `source code <https://github.com/srsran/srsRAN_project>`_.
    * Learn about the team behind srsRAN Project on our `company website <www.srs.io>`_.
    * Go to the srsRAN Project `GitHub Discussions <https://github.com/srsran/srsran_project/discussions>`_ for news and user support.
